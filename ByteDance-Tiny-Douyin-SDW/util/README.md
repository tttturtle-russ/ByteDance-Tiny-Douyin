## 抖音极简版后端API
## TikTok Tiny Backend API
***
### 项目架构
        |-- controller  // 封装api业务逻辑
        |-- dao         // 数据库交互
        |-- idl         // proto文件所在处
        |-- middleware  // 中间件
        |-- model       // 定义模型
        |-- client      // 管理数据库有关，mysql,redis等
        |-- routers     // 定义路由
        |   |-- routers.go
        |-- service     // 封装通用的逻辑
        |-- test        // 单元测试
        |-- util        // 工具类代码
        |-- .gitignore
        |-- README.md
        |-- go.mod
        |-- go.sum
        |-- main.go     // 程序入口
